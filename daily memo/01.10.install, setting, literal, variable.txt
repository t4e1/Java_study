JDK / JRE

왜 java를 사용하는가? 설명할 수 있어야함

jvm : 우리가 만든 java파일을 OS가 이해할 수 있도록 마지막으로 변환해주는 역할 
jre : 만든 파일을 실행하기 시키는 것
jdk : 실제 개발 환경. 개발에 필요한 도구와 컴파일러가 있음. JDK는 유료, 나머지는 무료 ( 개발할 필요 없으면 jre까지만 설치하고 무료로 사용 가능 )

컴파일 에러 vs 런타임 에러 
 컴파일 에러 : 실행 시키기 전 문법적으로 오류가 날 경우 (빨간 줄이 생김) 
 런타임 에러 : 문법적으로는 문제가 되지 않지만 실행시키면 에러가 나는 경우
